- 그동안 회사에서 진행했던 데이터 수집 프로젝트는 아래와 같습니다.
    - 공정거래위원회
    - 근로복지공단
    - DART
    - 국토교통부


- 단순한 웹 스크래핑에 그치지 않고, 사용자가 쉽게 사용할 수 있도록 GUI Programming을 설계했습니다.
- 프로그램이 실행되어 원하는 조건을 입력해서 수집 버튼을 누르면 자동으로 엑셀 파일이 저장됩니다.
- 저장된 프로그램은 텔레그램 봇이나 메일로 자동화되어 보내집니다.
